# Contributing

Contributions are welcome! Here's how to get started.

## Setup

```bash
git clone https://github.com/paulosergio-jnr/aws-whats-new-mcp.git
cd aws-whats-new-mcp
uv pip install fastmcp httpx
```

## Making changes

1. Fork the repo and create a branch from `main`
2. Make your changes and ensure `server.py` runs without errors
3. Open a pull request with a clear description of what you changed and why

## Reporting issues

Open a [GitHub issue](https://github.com/paulosergio-jnr/aws-whats-new-mcp/issues) with steps to reproduce.
